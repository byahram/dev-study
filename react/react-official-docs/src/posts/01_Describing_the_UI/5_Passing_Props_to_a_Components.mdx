# 컴포넌트에 Props 전달하기

## ◾️ Props란?

- props는 컴포넌트에 데이터를 전달하는 방법이다. (부모 컴포넌트 -> 자식 컴포넌트)
- 컴포넌트를 재사용 가능하게 만들 수 있다.
- 객체, 배열, 함수를 포함한 모든 JavaScript 값을 전달할 수 있다.

```text
function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}
```

### 1. 컴포넌트에 props 전달하기

- Avatar에 person (객체)와 size (숫자)를 전달

```text
export default function Profile() {
  return (
    <div>
      <Avatar
        size={100}
        person={{
          name: 'Katsuko Saruhashi',
          imageId: 'YfeOqp2'
        }}
      />
      <Avatar
        size={80}
        person={{
          name: 'Aklilu Lemma',

        }}
      />
      <Avatar
        size={50}
        person={{
          name: 'Lin Lanying',

        }}
      />
    </div>
  );
}
```

### 2. 자식 컴포넌트 내부에서 props 읽기

```text
import { getImageUrl } from './utils.js';

function getImageUrl(person, size = 's') {
  return (
    'https://i.imgur.com/' +
    person.imageId +
    size +
    '.jpg'
  );
}

function Avatar({ person, size }) {
  return (
    <img
      className="avatar"
      src={getImageUrl(person)}
      alt={person.name}
      width={size}
      height={size}
    />
  );
}
```

## ◾️ prop의 기본값 지정하기

값이 지정되지 않았을 떄 prop에 기본값을 주길 원한다면 변수 바로 뒤에 `=`과 함께 기본값을 넣어준다
