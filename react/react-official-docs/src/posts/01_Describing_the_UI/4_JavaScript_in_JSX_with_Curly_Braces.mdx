# 중괄호가 있는 TSX에서 자바스크립트 사용하기

JSX에서는 `{}` 중괄호를 사용하여 JavaScript 표현식을 삽입할 수 있다.

## ◾️ 따옴표로 문자열 전달하기

- 문자열 속성을 JSX에 전달할 때는 큰따옴표(`"`)나 작은따옴표(`'`)를 사용한다.

  ```text
  export default function Avatar() {
    return (
      <img
        className="avatar"
        src="https://i.imgur.com/7vQD0fPs.jpg"
        alt="Gregorio Y. Zara"
      />
    );
  }
  ```

## ◾️ 중괄호 사용하기: JSX 내부에서 JavaScript 사용하기

- 속성 값에 변수를 전달하려면 중괄호 {}를 사용한다.
- `src={avatar}`는 avatar 변수를 읽지만 `src="{avatar}"`는 `{avatar}` 문자열을 전달

  ```text
  export default function Avatar() {
    const avatar = "https://i.imgur.com/7vQD0fPs.jpg";
    return (
      <img
        className="avatar"
        src={avatar}
        alt="Gregorio Y. Zara"
      />
    );
  }
  ```

- JSX 태그 내부에서도 중괄호로 JavaScript 표현식을 삽입할 수 있다.

  ```text
  export default function TodoList() {
    const name = "Gregorio Y. Zara";
    return <h1>{name}'s To Do List</h1>;
  }
  ```

- 함수 호출도 가능하다.

  ```text
  const today = new Date();

  function formatDate(date) {
    return new Intl.DateTimeFormat("en-US", { weekday: "long" }).format(date);
  }

  export default function TodoList() {
    return <h1>To Do List for {formatDate(today)}</h1>;
  }
  ```

## ◾️ 이중 중괄호 사용하기: 객체를 속성으로 전달

- 객체를 속성으로 전달할 때는 중괄호 안에 또 다른 객체 리터럴을 넣기 때문에 중괄호를 두 번 사용한다.
- 인라인 스타일 속성은 객체로 전달하며, 속성명은 camelCase로 작성한다.

  ```text
  export default function TodoList() {
    return (
      <ul
        {/* 인라인 style 프로퍼티는 캐멀 케이스로 작성 */}
        style={{
          backgroundColor: "black",
          color: "pink",
        }}
      >
        <li>Improve the videophone</li>
        <li>Prepare aeronautics lectures</li>
        <li>Work on the alcohol-fuelled engine</li>
      </ul>
    );
  }
  ```

- 객체를 변수로 선언해 전달할 수도 있다.

  ```text
  const person = {
    name: "Gregorio Y. Zara",
    theme: {
      backgroundColor: "black",
      color: "pink",
    },
  };

  export default function TodoList() {
    return (
      <div style={person.theme}>
        <h1>{person.name}'s Todos</h1>
        <img
          className="avatar"
          src="https://i.imgur.com/7vQD0fPs.jpg"
          alt="Gregorio Y. Zara"
        />
        <ul>
          <li>Improve the videophone</li>
          <li>Prepare aeronautics lectures</li>
          <li>Work on the alcohol-fuelled engine</li>
        </ul>
      </div>
    );
  }
  ```
